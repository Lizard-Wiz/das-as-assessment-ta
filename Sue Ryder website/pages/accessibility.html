Accessibility

We strive for sueryder.org and its features to be built in line with the World Wide Web Consortium's WCAG 2.1 guidelines, level AA. If, however, you find that there is some content or a feature on the site that you are unable to use, please let us know using the form below. We will consider all feedback carefully to make improvements where we can.

We are also currently auditing our website as we are aiming to meet the new WCAG 2.2 Guidelines, which was released in October 2023.

If you are having trouble using the website and you need help with an enquiry quickly, please call us on 0808 164 4572 or email us at web.enquiries@sueryder.org

Help with using this website

For guidance on how to make it easier to use this website on your device – such as changing colours or font sizes, using your keyboard instead of a mouse and magnifying the display – you can find a range of useful tips on AbilityNet's My Computer My Way page.
